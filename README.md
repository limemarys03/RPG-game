# Old Mansion - Atmospheric Text RPG

A dark, atmospheric text-based RPG quest developed in Python. The game challenges players to explore a mysterious old mansion, manage an inventory, and crack a secret password based on a song lyric.

## 🌟 Features
* **Deep Atmosphere**: Immerse yourself in a rainy autumn vibe with eerie rooms.
* **Interactive Exploration**: Five unique locations to visit (Corridor, Living Room, Kitchen, Locked Room, Balcony).
* **Inventory System**: Find and collect tools and weapons like a rusty key, a sharp knife, or a scythe.
* **Riddle Mechanics**: Puzzle-locked room that requires a correct password hint to enter.
* **Classic Architecture**: Built using explicit control flow structures inspired by retro text adventures.

## 🛠️ Tech Stack & Requirements
* **Language**: Python 3.x
* **Libraries**: `time`, `os`, `sys` (built-in)

## 🚀 How to Run
1. Clone this repository or download the source code.
2. Open your terminal or a mobile compiler (like Pydroid 3).
3. Run the script:
   ```bash
   python main.py
   ```
4. Enter numbers from 1 to 5 to make your choices and explore the mansion.
